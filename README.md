# Our Recipes

"Our Recipes" is a full-stack project developed during [Ironhack's](https://www.ironhack.com/) Web Development course  (module 2) where the goal was to apply the backend concepts learned during that period.

### Tech
- Node.Js
- Express.Js
- MongoDB
- Mongoose
- Handlebars 
- Javascript ES6
- HTML5
- CSS3
- Bootstrap 
- Git
